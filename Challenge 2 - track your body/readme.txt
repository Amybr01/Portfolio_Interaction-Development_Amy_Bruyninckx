# Interactive Mini-Game with PoseNet

This mini-game allows users to interact with objects on the screen using PoseNet to track hand movements.

## Online Demo

Check out the online demo [here](codesandbox!!).

## Setup

1. Clone the repository.
2. Open `index.html` in your browser.

## Technologies Used

- p5.js
- ml5.js (PoseNet)
- HTML, CSS, JavaScript

## How to Play

- Move your hands in front of the camera to interact with the objects on the screen.
- Poke, pick up, or push the objects to see the effects.
